# Legal AI Starter

Launch your own legal AI in 5 minutes.

## Setup

1. Create entity on [Subfeed Cloud](https://cloud.subfeed.app)
2. Copy API key + Entity ID
3. Clone this repo
4. `cp .env.example .env` and paste credentials
5. `npm install && npm run dev`
