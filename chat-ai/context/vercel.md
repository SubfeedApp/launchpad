# Subfeed Chat AI - Vercel Deployment

## One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/SubfeedApp/launchpad/tree/main/chat-ai&env=SUBFEED_API_KEY,SUBFEED_ENTITY_ID&envDescription=Get%20these%20from%20cloud.subfeed.app)

## Manual Setup

1. Fork/clone this repo
2. Connect to Vercel
3. Add environment variables:
   - `SUBFEED_API_KEY`
   - `SUBFEED_ENTITY_ID`
4. Deploy
